# تطبيق الموظف الفني (AppEmp)

تطبيق Flutter لموظف روز — مهام + إشعارات FCM + تليجرام + تحديث فوري عبر SignalR.

## الهوية والواجهة

- شعار التطبيق: خلفية بيضاء + شعار روز
- شريط تنقل سفلي:
  1. المهام
  2. بالانتظار
  3. مكتملة
  4. الإعدادات (الاسم / الشركة / اسم المستخدم / ربط تليجرام)

## إشعارات الآيفون / الويب

على الآيفون مع ويب التطبيق، اربط حسابك من **الإعدادات → ربط تليجرام**.
بعد الضغط على Start في البوت `rozsysnet_bot` ستصلك إشعارات المهام هناك.
